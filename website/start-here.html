Your Simply Plural export may be the most complete copy of your System data now. Keep it private, keep a backup, and keep it somewhere you control while PluralBridge builds the first browser-based release.

What PluralBridge is building now

  • A free, open-source browser-based release for plural Systems affected by the Simply Plural shutdown.
  • A safe path to review a Simply Plural export before anything is saved into PluralBridge.
  • A place to view, add, and update member profiles after preserved data has somewhere to land.

What comes next

  • PluralBridge is working toward the first browser-based release.
  • The import path will let users review a Simply Plural export before saving it.
  • Users should keep exported files private and backed up while that work continues.

If you missed the export window

PluralBridge cannot promise recovery of data that was never exported. The project is focused on helping preserved data move forward safely. Watch the public site and project updates for any new information, and avoid sending private account details, screenshots, or System data through email, public threads, or direct messages.

What to do now

If you have a Simply Plural export, keep it private and backed up. PluralBridge is building the first browser-based release so Systems can review preserved data before saving it, then work with member profiles in a clearer place.

Where to go next

If you already exported, start with safety. If you want to follow what PluralBridge is building, start with the browser release page. If you want to help build, review, test, document, or think through security and privacy, start with Help Build.
