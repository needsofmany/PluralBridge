Privacy reminder: Simply Plural exports may contain sensitive account, token, note, message, friend, avatar, and System data. Treat export files as private backups. Do not post them publicly or attach them to support requests.

Simply Plural data preservation

Get your data out while the bridge is still standing.

PluralBridge is a preservation and continuity project for plural Systems that need a practical way to export Simply Plural data into local JSON files, database tables, reports, viewers, and future tools.

Export first. Refine later.

As currently announced, the Simply Plural servers are expected to shut down on July 1, 2026, with user data and avatars permanently deleted after shutdown. The first job is getting user-owned data safely out while the original service still exists.

What PluralBridge is building toward

The current export scripts are the first layer. The longer path includes local viewers, portable databases, cloud migration paths, compatible APIs where feasible, and future clients that can make preserved plural System data useful after the original service disappears.

PluralBridge is an independent project by Needs of the Many.

PluralBridge is not affiliated with Simply Plural or Apparyllis.
