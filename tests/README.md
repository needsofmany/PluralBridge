# Tests

This directory is reserved for PluralBridge tests.

Tests should use synthetic or redacted data only.

Do not place real Simply Plural exports, avatar images, notes, API tokens, screenshots, SQL Server backups, or generated private reports in this directory.

Future tests may cover:

- JSON shape parsing
- API response handling
- export path handling
- SQL Server load-script validation
- avatar URL construction
- privacy/leak checks
