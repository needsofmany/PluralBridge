Task 24 expanded diagnostic trace forbidden facts

Purpose:
  Define the content that Expanded Diagnostic Trace must never emit.

Trace mode:
  Expanded Diagnostic Trace is Level 2 trace.
  Level 2 trace is optional.
  Level 2 trace is controlled by a future flag.
  Level 2 trace remains diagnostic-only.

Forbidden Level 2 facts:
  email addresses
  display names
  member names
  note text
  custom field values
  avatar data
  imported raw payloads
  exception detail containing user data
  request body content
  response body content

Forbidden derived content:
  transformed user notes remain forbidden
  summarized user notes remain forbidden
  imported source payload fragments remain forbidden
  exception messages remain forbidden when they may contain user data

Done when forbidden facts include:
  email addresses
  display names
  member names
  note text
  custom field values
  avatar data
  imported raw payloads
  exception detail containing user data
  request body content
  response body content
