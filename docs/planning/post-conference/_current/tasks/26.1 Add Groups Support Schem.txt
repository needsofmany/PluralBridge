Task 26 is **Add groups support schema**. This task adds database support for system-scoped groups and group-member links. Groups preserve Simply Plural organization data, but do not become Account, System, Membership, Role, or a security boundary.

26.1  Define groups support intent
      Add the Task 26 note that defines groups as system-scoped organization.
      Done when:
        groups are named as system-internal organization
        groups are described as data/import support
        groups are not described as security boundaries
        pb_groups is named
        pb_group_members is named

26.2  Add groups schema design record
      Add a design file that defines the group schema shape.
      Done when it states:
        pb_groups stores one row per group
        pb_group_members stores one row per member assignment to a group
        group membership is many-to-many
        a member can belong to more than one group
        groups belong to one SystemId
        groups do not replace System or Membership
