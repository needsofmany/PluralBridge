Task 24 expanded diagnostic trace allowed facts

Purpose:
  Define the additional non-sensitive facts that Expanded Diagnostic Trace may emit in the future.

Trace mode:
  Expanded Diagnostic Trace is Level 2 trace.
  Level 2 trace is optional.
  Level 2 trace is controlled by a future flag.
  Level 2 trace remains diagnostic-only.

Allowed Level 2 facts:
  controller name
  action name
  route pattern
  safe table family name
  row count
  resolved SystemId
  resolved SystemMembershipId
  authorization reason category
  feature flag name
  elapsed timing

Allowed reason categories:
  account_not_resolved
  membership_not_resolved
  current_system_not_resolved
  membership_status_inactive
  membership_system_mismatch
  membership_id_mismatch
  authorized

Done when allowed facts include:
  controller name
  action name
  route pattern
  safe table family name
  row count
  resolved SystemId
  resolved SystemMembershipId
  authorization reason category
  feature flag name
  elapsed timing
