# PB-NEXT Reference Material

This folder contains PB-NEXT planning source material and reference documents.

The active control document is the PB-NEXT Workflow document. These Markdown files are supporting material, not the execution roadmap.

## Folder Map

- `00-crosswalk` — blueprint crosswalk and high-level source mapping.
- `10-core-spine` — SystemId, AccountId, membership, relationship, ReBAC, and user-model planning.
- `20-auth-privacy-consent` — authorization, authorization shell, consent, and consent-schema planning.
- `30-audit` — audit model and audit-schema planning.
- `40-import-upload` — dry-run, import, import job ledger, and upload intake planning.
- `50-implementation-control` — implementation queue, gate, and Alpha commit planning.

## Use Rule

Use the PB-NEXT Workflow document to decide what to build next.

Use this folder only when a workflow chapter needs supporting detail from prior planning.
